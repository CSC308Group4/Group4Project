Grocery List: Keep track of items in your fridge and which ones you need to buy
Elijah Moody, Hunter Jackson, Nicolai Arnett

Implemented Concepts:

Optionals and Optional Binding: Using optionals in Item class.

Functions and Closures: View controller contains functions for readability and optimization. Closures are used in array manipulation.

Syntax Optimization: Closure syntax was optimized at multiple locations.

TableView with delegate: Main storyboard had a table view. A delegate was used to pass data through segue.

AutoLayout: Auto layout was used to maintain UI structure across devices.

Navigation Controller: A navigation controller was used for titles and segues.

View Controller LifeCycle: Multiple stages of the view controller lifecycle were used in the view controllers to prepare the next screen and smooth flow.

Input Validation: Using guard we validated input and returned if not ideal.

Type Casting: Used type casting to convert strings into Integers for computation.

Classes: Created an Item class to keep data together for the lists.


Instructions:

1. Open app
2. Add an item
3. Remove some
3. Go to grocery list
4. Change grocery list
5. Return, repeat If necessary
